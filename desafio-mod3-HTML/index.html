Este é meu site estático de estudos: pratiquei Amazon S3 (hospedagem estática), EC2, EBS e integrações básicas.

Treinamento

Conteúdo prático do bootcamp com foco em fundamentos de Cloud na AWS:

  • Visão geral de serviços: EC2, S3, EBS, IAM e VPC.
  • Hospedagem estática em S3 e noções de CDN (CloudFront) para distribuição.
  • Provisionamento básico de instâncias EC2 e anexação de volumes EBS.
  • Conceitos de segurança (credenciais, políticas IAM) e boas práticas iniciais.
  • Git/GitHub para versionamento e publicação de conteúdo estático.

Habilidades

Em desenvolvimento:
  • Amazon EC2 (máquinas virtuais, tipos de instância e uso básico).
  • Amazon S3 (armazenamento de objetos, sites estáticos e versionamento).
  • Amazon EBS (volumes persistentes para EC2: criação, anexação e snapshots).
  • Fundamentos de redes na AWS (VPC, sub-redes públicas/privadas — noções).
  • Git e GitHub para controle de versões e deploy estático.
